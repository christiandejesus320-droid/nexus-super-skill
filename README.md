# 🚀 NEXUS Super Skill: El Motor de Ejecución Implacable

### ¿Qué es NEXUS Super Skill?
Es un sistema operativo de alto rendimiento diseñado para **Claude Code**. Mientras que otros usan la IA para "chatear", nosotros la usamos para **ejecutar**. Esta skill fusiona persistencia de datos, gestión de objetivos críticos y un sistema de bloqueo (*Stop Gate*) que garantiza que ninguna tarea se quede a medias.

### ¿Cómo ayuda a los demás?
NEXUS Super Skill elimina los tres problemas más grandes del trabajo con IA:
1. **La Amnesia**: Gracias a su base de datos SQLite, la IA recuerda tus objetivos incluso si cierras la sesión.
2. **La Dispersión**: El comando `/loop` obliga a la IA a seguir un camino de: Diagnóstico → Acción → Evidencia → Verificación.
3. **La Incompletitud**: El *Stop Gate* impide que la IA se detenga si aún hay trabajo pendiente que genera valor.

### ¿Por qué es diferente?
Está construida bajo la metodología de **NEXUS LABS**. No aceptamos teoría sin código, ni planes sin resultados. Ayuda a emprendedores y desarrolladores a pasar de la idea al producto final en tiempo récord, operando todo desde una simple laptop.

## 🛠️ Comandos de Poder

*   **/goal [objetivo]**: Define tu misión. Activa el modo de alta prioridad.
*   **/loop**: Inicia el ciclo de trabajo. Planifica, ejecuta y entrega pruebas reales.
*   **/nexus status**: Auditoría total de tu progreso y presupuesto de tokens.
